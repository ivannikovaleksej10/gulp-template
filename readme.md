# GULP
My gulp template.

## Preparing for work
1. ` $ git clone ... `
2. ` $ cd gulp `
3. ` $ npm i `
4. ` $ gulp `

## Commands
- ` $ gulp ` // dev build (./build)
- ` $ gulp docs ` // prod build (./docs(easy to host in github pages))

## Other
- babel & browser version settings in package.json
- build settings in gulp dir
